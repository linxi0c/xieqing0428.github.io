Hbase：我不认识Hive…

Hbase 安装配置

Hbase

安装Hbase

  • cd /mnt/hgfs/Hadoop

  • cp hbase-1.3.3-bin.tar.gz /usr/local/src/

  • cd /usr/local/src/

  • tar zxvf hbase-1.3.3-bin.tar.gz

  • rm -rf hbase-1.3.3-bin.tar.gz

Hbase环境变量：

  • vim ~/.bashrc
  • source ~/.bashrc

修改Hbase配置

  • cd hbase-1.3.3

创建临时目录和文件目录：

  • mkdir logs

  • mkdir tmp

配置文件：

  • cd conf

  • vim regionservers

  • vim hbase-env.sh
  • vim hbase-site.xml

启动集群

仅在Master

启动Hbase服务：

  • start-hbase.sh

关闭Hbase服务：

  • stop-hbase.sh
